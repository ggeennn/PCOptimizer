# PCOptimizer

一个用于 Windows PC 系统优化、安全维护和软件包管理的工具集。

## 当前功能

1. **软件包管理**
   - 导出已安装软件列表
   - 安装推荐优化软件
   - 自动检查更新

2. **系统维护**
   - 自动系统清理（CCleaner）
   - 定时清理计划
   - 系统性能监控

## 使用方法

1. **基础设置**：
   ```powershell
   # 安装必要的清理工具
   winget install Piriform.CCleaner
   ```

2. **自动清理**：
   ```powershell
   # 设置每3天自动清理
   .\GetPackages.ps1
   # 选择选项 7 设置清理计划
   ```
